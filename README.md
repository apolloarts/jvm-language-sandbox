# jvm-language-sandbox
A project to show simple examples for JVM languages such (Java, Kotlin and Scala) together in one place.

### Basic
- Hello World
- List
- Singleton
- Factory

### Scala specifics
- Scala Lambda
- Scala tail recursions
- Scala Closures
- Scala Traits
- Scala pattern matching
- Scala Implicits
- Scala symbols
